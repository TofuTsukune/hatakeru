# 畑管理アプリ 操作マニュアル

## はじめに

**畑管理アプリ** は、畑のレイアウト・作物・作業記録・予定をブラウザだけで管理できるアプリです。  
インターネット接続不要。データはこのブラウザに自動保存されます。

---

## 基本的な使い方の流れ

```
① 作物を登録する　→　② 畑マップに配置・日付を設定する　→　③ 作業を記録する
```

---

## 画面の見かた

画面上部のタブで切り替えます。

| タブ | できること |
|------|-----------|
| 📍 畑マップ | 畑のレイアウトを視覚的に管理する |
| 🥬 作物 | 作物を登録・編集する |
| 📋 作業ログ | 水やり・施肥などの作業を記録する |
| 📅 カレンダー | 植付け日・収穫予定・作業を月単位で確認する |

---

## 作物を登録する

### 新しい作物を追加する

1. 「🥬 作物」タブを開く
2. 右上の **「＋ 作物を追加」** をクリック
3. 以下を入力して **「保存」** をクリック

| 項目 | 説明 |
|------|------|
| 作物名 | 必須。例：トマト、ナス、キュウリ |
| 品種 | 任意。例：大玉トマト、千両二号 |
| ステータス | 予定 / 栽培中 / 収穫済み |
| カラー | 畑マップでの表示色（10色から選択） |
| メモ | 栽培の注意点など自由記入 |

> **ポイント:** 植付け日・収穫予定日は畑マップの区画ごとに設定します。ここでは入力しません。

### 作物を編集・削除する

- カードの **「編集」** ボタンをクリックして編集モーダルを開く
- 削除は編集モーダル内の **「削除」** ボタンから行う
- 削除すると、その作物に紐づく作業ログ・予定・区画割り当てもすべて削除されます

---

## 畑マップを使う

### グリッドの見かた

- 縦軸: A・B・C…（行）
- 横軸: 1・2・3…（列）
- セル左上の小さな文字（例: **A2**）が区画番号です

色がついているセルに作物が配置されています。  
日付が設定されている区画には `植:6/12`・`収:8/15` のように表示されます。

### 区画に作物を配置する

1. 配置したいセルをクリック
2. **「割り当て作物」** ドロップダウンで作物を選択
3. 作物情報編集フォームが展開される
4. 植付け日・収穫予定日を入力（任意）
5. 必要に応じて作物名・品種・ステータス・メモも編集できる
6. **「保存」** をクリック

> **例:** A2にトマトを6月12日、A3に同じトマトを6月13日に植え付けたい場合  
> → A2をクリック → トマトを選択 → 植付け日 `2026-06-12` → 保存  
> → A3をクリック → トマトを選択 → 植付け日 `2026-06-13` → 保存  
> 同じ作物でも区画ごとに日付を個別設定できます。

### 区画の作物を解除する

1. 解除したいセルをクリック
2. 「割り当て作物」で **「（なし）」** を選択
3. **「保存」** をクリック

### グリッドサイズを変更する

1. 「グリッドサイズ変更」ボタンをクリック
2. 列数（最大16）・行数（最大12）を入力
3. **「適用」** をクリック

> **注意:** 縮小すると、範囲外になった区画のデータは削除されます。

## 作業を記録する

### 作業ログを追加する

**方法1:** 「📋 作業ログ」タブ → **「＋ 記録を追加」**  
**方法2:** 「🥬 作物」タブのカード → **「作業記録」** ボタン（作物が自動入力される）

| 項目 | 説明 |
|------|------|
| 日付 | 必須 |
| 作業種別 | 💧水やり / 🌿施肥 / 🌾収穫 / 🌀草取り / 🌱植付け / 📝その他 |
| 作物 | 任意。対象の作物を選択 |
| メモ | 作業の詳細など |

### ログを絞り込む

作業ログ画面のフィルターで、作物別・作業種別で絞り込めます。

---

## カレンダーを使う

### 画面の見かた

月単位で以下の情報が表示されます:

| 表示 | 内容 |
|------|------|
| 🌱 植付け: トマト (A2) | 区画A2のトマトの植付け日 |
| 🌾 収穫予定: トマト (A3) | 区画A3のトマトの収穫予定日 |
| 💧 水やり（緑・薄） | 作業ログの記録 |
| タイトル文字（青） | 手動で追加した予定 |

- **◀ ▶** ボタンで月を移動します
- 今日の日付は青い丸数字で表示されます

### 予定を追加する

**方法1:** 「＋ 予定を追加」ボタンをクリック  
**方法2:** カレンダーの日付セルをクリック（日付が自動入力される）

| 項目 | 説明 |
|------|------|
| 日付 | 必須 |
| タイトル | 必須。例：トマトの追肥 |
| 種別 | 作業の種類 |
| 関連作物 | 任意 |

---

## よくある操作

### 同じ作物を複数区画に植える

同じ作物を選んでセルごとに異なる植付け日を設定できます。カレンダーでは区画番号（A2, A3など）付きで区別して表示されます。

### 作物の色を変えたい

「🥬 作物」→ 編集 → カラーを選び直して保存。

### データが消えてしまった

データはブラウザの `localStorage` に保存されています。以下の操作でデータが消えることがあります:
- ブラウザの「閲覧データを消去」
- シークレット/プライベートモードでの使用（タブを閉じると消える）
- 異なるブラウザ・端末からのアクセス（データは共有されない）

---

## 操作のヒント

- モーダル（入力画面）は、背景の暗い部分をクリックしても閉じられます
- 区画をクリックすると作物情報をその場で編集できるので、マップと作物管理を行き来する必要がありません
- カレンダーの予定は完了済みになると取り消し線で表示されます
